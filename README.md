# 📊 Sales Segmentation – RFM Analysis | Power BI

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** that performs **Sales Segmentation using the RFM (Recency, Frequency, Monetary) Analysis** technique. The dashboard helps businesses identify customer purchasing patterns, classify customers into meaningful segments, and support data-driven sales and marketing strategies.

---

## 🎯 Objective

The primary objective of this project is to analyze customer purchasing behavior using the RFM model and identify different customer segments based on:

- **Recency** – How recently a customer made a purchase.
- **Frequency** – How often a customer makes purchases.
- **Monetary** – How much revenue a customer generates.

This analysis helps businesses improve customer retention, personalize marketing campaigns, and increase sales.
